# HACS Integrations

Custom integrations installed via Home Assistant Community Store (HACS).

## Alexa Media Player
- **Repo:** alandtse/alexa_media_player
- **Version:** 5.15.0
- **Purpose:** Control and push notifications to Amazon Echo devices from Home Assistant. Enables TTS announcements, do not disturb control, and media player entities for all Alexa devices on the network.
- **Devices:** Amazon Echo Show 10, Echo Spot, Fire TV Sticks (x3), Samsung QLED TV

## Why HACS?
These integrations are not available in the official Home Assistant integration store. HACS provides a curated community store for custom components that extend HA's capabilities beyond the core offering.
